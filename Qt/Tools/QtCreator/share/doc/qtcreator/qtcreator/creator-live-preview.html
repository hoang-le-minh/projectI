Building and Running ◦ Previewing on Desktop

Validating with Target Hardware

You can use the live preview feature to preview a UI file or the entire UI on the desktop, as well as on Android and embedded Linux devices. The changes you make to the UI are instantly visible to you in the preview.

In addition, you can use Qt Design Viewer to run Qt Quick UI projects in most widely-used web browsers on the desktop and in mobile devices This enables you to easily share your designs with reviewers who don't have Qt Creator.

  • Previewing on Desktop

    You can preview individual QML files or the whole UI.

  • Previewing on Devices

    You can preview Qt Quick applications on devices that you have connected to the development PC. For more information, see Connecting Devices.

  • Previewing in Browsers

    You can open Qt Design Viewer in a browser and load applications to it.
