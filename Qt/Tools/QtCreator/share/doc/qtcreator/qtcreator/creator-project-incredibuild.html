Setting Up IncrediBuild

IncrediBuild accelerates process execution and thus shortens the time you spend on building C++ code. In addition, you can view the build progress in the graphical Build Monitor.

The IncrediBuild plugin is delivered with Qt Creator. For more information, see IncrediBuild for Qt Creator.

To use IncrediBuild, install IncrediBuild Agent on the development host. Then specify IncrediBuild build steps for your project. For more information, see IncrediBuild Build Steps.
